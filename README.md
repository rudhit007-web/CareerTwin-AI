# CareerTwin AI

CareerTwin AI is an AI-powered career guidance platform built using IBM watsonx.ai, React, FastAPI, and Python.

## Features

- User Authentication
- Resume Upload
- Resume Analysis using AI
- ATS Feedback
- Skill Gap Analysis
- Personalized Learning Roadmap
- AI Career Mentor Chatbot
- Career Progress Tracking

## Tech Stack

Frontend
- React
- Vite
- Tailwind CSS

Backend
- FastAPI
- Python

AI
- IBM watsonx.ai
- Meta Llama 3.3 70B Instruct

## Installation

### Backend

```bash
cd backend
pip install -r requirements.txt
python -m uvicorn main:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
